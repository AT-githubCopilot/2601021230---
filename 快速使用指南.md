# 快速使用指南

## 环境准备

1. 安装Python 3.8及以上版本
2. 安装依赖库：
   ```bash
   pip install -r requirements.txt
   ```

## 运行程序

```bash
python main.py
```

## 基本操作流程

### 步骤1：导入视频
- 点击「导入视频」按钮，选择一个MP4视频文件
- 或直接拖拽MP4文件到程序窗口
- 系统会自动解析视频信息并显示

### 步骤2：设置提取参数
- 在「关键帧提取设置」区域：
  - 设置「提取数量」（2-100张）
  - 选择「图片格式」（JPG/PNG）
  - 设置「图片质量」（仅JPG有效，1-100）
  - 选择「保存路径」

### 步骤3：提取关键帧
- 点击「提取关键帧」按钮
- 等待提取完成
- 右侧预览区会显示提取的关键帧

### 步骤4：设置宫格参数
- 在「宫格合成设置」区域：
  - 选择「宫格布局」（自动计算或手动模板）
  - 设置「图片间距」（0-20像素）
  - 设置「边框宽度」（0-10像素）

### 步骤5：合成宫格图
- 点击「合成宫格图」按钮
- 等待合成完成
- 查看合成结果并保存

## 功能说明

### 视频信息显示
- 显示视频文件名、分辨率、时长、帧率等信息
- 帮助用户了解视频基本属性

### 关键帧提取模式
- **均匀间隔模式**：按指定数量均匀提取关键帧，确保首尾帧必包含
- **关键场景模式**：基于画面变化程度自动识别关键帧（开发中）
- **时间点指定模式**：用户指定具体时间点进行截取（开发中）
- **混合模式**：结合均匀间隔和关键场景识别（开发中）

### 宫格合成选项
- **自动计算布局**：根据图片数量自动计算最优宫格布局
- **手动模板**：支持2×2、3×3、4×4等常用模板
- **图片适配模式**：
  - 中心裁剪：保持画面主体，裁剪边缘
  - 保持纵横比：保留完整画面，可能有黑边

## 常见问题

### 问题1：无法导入视频
- 检查视频格式是否为MP4
- 检查视频文件是否损坏
- 检查视频编码是否被支持

### 问题2：提取关键帧速度慢
- 视频分辨率过高或文件过大
- 提取数量过多
- 电脑性能不足

### 问题3：合成的宫格图不清晰
- 原始视频分辨率较低
- 图片质量设置过低
- 宫格布局不合理导致图片压缩过度

## 快捷键

- 无特殊快捷键，所有操作均通过界面按钮完成

## 输出文件

- **关键帧图片**：保存到指定路径，命名格式为「视频文件名_序号.格式」
- **宫格图**：保存到指定路径下的「宫格图」文件夹，命名格式为「视频文件名_宫格图.jpg」
